# lever-offline — Design Context

## Users

Mongolian students taking a 3-hour IELTS mock exam. High-stakes, single sitting, focused. Typical context: quiet room, laptop, fullscreen, 180 minutes of sustained reading + listening + writing. They've paid real money for this. They want the experience to feel like the real test, but *better* — calmer, more trustworthy, less noisy than typical CD-IELTS clones.

**Job to be done:** give me a credible, confidence-building rehearsal of the real IELTS experience so I show up on exam day feeling prepared.

**Emotional goal:** *"you've got this."* Calm confidence, not anxiety or stimulation. Quiet focus during sections; small moments of delight at natural breakpoints (passage turns, section transitions).

## Brand Personality

**Three words:** confident, trustworthy, education.

The brand identity is a black "Lever edu" wordmark with a mint-green page-curl at the bottom-right corner. The page-curl is the signature motif — paper-as-affordance.

## Aesthetic Direction

**Theme:** light mode. Paper feel. Long reading sessions + low-light-sensitivity + the entire brand metaphor is paper, so light is the correct choice, not a default.

**Direction:** editorial-minimal. Think *a beautifully typeset book crossed with a well-made product*. High typographic contrast. Generous margins. Mint used sparingly as accent — it appears, it matters, then it gets out of the way.

**Reference feel:** the restraint of Readwise Reader, the paper-surface of NYT print editions, the typographic confidence of Stripe docs. Page-curl as the one whimsical detail.

**Anti-references:**
- **NOT Duolingo** — no gamification, no cartoon mascots, no cheeky illustrations, no chunky shadowed buttons, no multi-color chaos. We are not celebrating streaks.
- **NOT generic SaaS** — no royal-blue primary, no drop-shadowed cards stacked on pastel backgrounds, no "Get started for free" energy.
- **NOT Cambridge-official** — not institutional-stiff or bureaucratic-serif. Confident, not starchy.

## Palette (OKLCH)

```
--ink:      oklch(0.18 0.005 264);   /* #0A0A0B — near-black, slight cool tint */
--ink-soft: oklch(0.32 0.006 264);   /* secondary text */
--muted:    oklch(0.62 0.008 264);   /* tertiary / borders / timestamps */
--rule:     oklch(0.89 0.008 264);   /* hairline dividers */
--paper:    oklch(0.98 0.008 85);    /* #FAF8F3 — warm off-white surface */
--paper-2:  oklch(0.96 0.010 85);    /* slightly deeper card surface */
--mint:     oklch(0.78 0.18  155);   /* #34DFA0-ish — brand accent, sampled from logo */
--mint-ink: oklch(0.38 0.12  155);   /* for mint-on-paper text if ever needed */
```

Neutrals are tinted warm (yellow-gold hue 85) toward the paper surface; ink is tinted slightly cool to read as "ink on paper" rather than cold gray. 60% paper / 30% ink + ink-soft / 10% mint.

## Typography

**Display — Literata** (Google Fonts): a contemporary book face designed by Google for long-form reading. Used for: passage titles, section headings, results page band-score numerals, any hero moment. Supports Cyrillic. Chosen because this product IS about reading and education — the typeface literally exists to serve that job. Not in the reflex-ban list.

**Body / UI — Hanken Grotesk** (Google Fonts): humanist sans, confident without being cold, trustworthy without being institutional. Used for: all UI chrome, questions, buttons, labels, body paragraphs inside passages (when prose-serif isn't right). Supports Cyrillic — critical for the Mongolian student-facing text.

**Scale** (fixed rem, product UI — no fluid-clamp in app chrome):
- 3rem / 1.05 / display (passage titles)
- 1.875rem / 1.15 / h2
- 1.25rem / 1.35 / h3
- 1.0625rem / 1.6 / body (reading prose)
- 0.9375rem / 1.5 / UI default
- 0.8125rem / 1.4 / labels, metadata
- 0.6875rem / 1.3 / microcopy, uppercase tags

Passage prose caps at 68ch. Display titles may go wider.

## Motif — the Page Curl

The single memorable detail. Used in exactly three places (rare = powerful):

1. **Decorative corner** on the active reading-passage card. A mint triangular fold on the bottom-right, echoing the logo.
2. **Transition animation** when moving between reading passages (1→2→3) and between sections (L→R→W). 3D `rotateY` with paper shadow + slight perspective. Feels like turning a physical page. ~450ms cubic-bezier deceleration.
3. **Brand mark** in the fullscreen header — tiny Lever-edu logo top-left, never explained, just present.

The curl is not used as decoration elsewhere. Overuse would kill its power.

## Design Principles

1. **Paper first, pixels second.** Treat the canvas as a page. Light mode, warm neutrals, restraint with color, generous margins, type that respects long reading.
2. **Mint is earned.** Accent green appears only on: the active tab indicator, primary CTAs, the passage-curl corner, answered-question ticks. Everywhere else is ink + paper.
3. **Typography carries hierarchy.** Use weight + size contrast, not boxes and shadows. Cards only where a real container is justified; most surfaces are just type on paper.
4. **Motion at breakpoints, not everywhere.** One deliberate page-turn transition > a dozen hover fades. Silence during focused work; delight at section boundaries.
5. **Quiet confidence.** The UI never raises its voice. Buttons aren't aggressively colored, timers aren't red-alert, errors aren't shouty. Students are already stressed; the interface does not add to it.

## Constraints

- Stack: Next.js 15 + Tailwind v3 + Radix primitives + Framer Motion (to be added for page-turn)
- Accessibility: WCAG AA contrast on all text; full keyboard nav for navigation/highlighting; respect `prefers-reduced-motion` — page-turn animation must degrade to a 180ms opacity crossfade when user has motion reduction on.
- Cyrillic support mandatory on every font chosen.
- Fullscreen exam layout must not visually shift when the fullscreen prompt is accepted.

## Do / Don't — quick reference

**Do**
- Use ink black for primary type, tinted grays for secondary.
- Use mint exactly where it's earned (CTA, active tab, curl, success tick).
- Let passages breathe — warm paper surface, generous leading, max 68ch measure.
- Animate passage/section transitions as physical page-turns.

**Don't**
- Use royal blue, cyan, or any AI-palette tells.
- Put icons-above-headings unless they carry meaning.
- Wrap every section in a drop-shadowed card.
- Use uppercase on long strings. Uppercase is for ≤2-word labels only.
- Use gradient text or >1px side-stripe borders. Ever.
